1. 문자열 데이터 타입
```
*&---------------------------------------------------------------------*  
*& Report Z02_06  
*&---------------------------------------------------------------------*  
*& 문자 데이터 타입  
*&---------------------------------------------------------------------*  
REPORT Z02_06.  
  
DATA: gv_f0.  
DATA: gv_f1 TYPE c.  
DATA: gv_f2(1) TYPE c.  
DATA: gv_f3(2) TYPE c.  
DATA: gv_f4 TYPE c LENGTH 2.  
DATA: gv_f5(5).  
  
gv_f0 = 'korea'.  
gv_f1 = 'korea'.  
gv_f2(1) = 'korea'.  
gv_f3(2) = 'korea'.  
gv_f4 = 'korea'.  
gv_f5 = 'korea'.  
  
  
WRITE: / gv_f0,  
/ gv_f1,  
/ gv_f2(1),  
/ gv_f3(2),  
/ gv_f4,  
/ gv_f5.  
  
  
DATA(gv_len) = STRLEN( gv_f5 ).  
WRITE: / gv_len.
```
